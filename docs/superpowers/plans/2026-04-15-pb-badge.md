# PB Badge Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Show a "PB!" badge on an exercise card when the user completes all sets at a weight higher than any previously recorded weight for that exercise+gym.

**Architecture:** Add `getPB`/`setPB` to `storage.js` (mirrors the existing `getWeight`/`setWeight` pattern). Add a pure `checkPB()` function to `app.js` that reads/updates the PB record and returns whether a badge should be shown. Wire `checkPB()` into `onStopwatchTap()` on the `done` transition, injecting a `.pb-badge` span into the card's title row. Style the badge in `style.css`.

**Tech Stack:** Vanilla JS, localStorage, Jest + jsdom for tests.

- [ ] **Step 3: Implement `getPB` and `setPB` in `storage.js`**

Add the following two functions to `storage.js`, right before the `if (typeof module !== 'undefined')` line at the bottom:

```js
function _pbKey(exerciseName, gymIndex) {
  return `pb__${exerciseName}__${gymIndex}`;
}

function getPB(exerciseName, gymIndex) {
  const raw = localStorage.getItem(_pbKey(exerciseName, gymIndex));
  if (!raw) return null;
  try {
    return JSON.parse(raw);
  } catch {
    return null;
  }
}

function setPB(exerciseName, gymIndex, data) {
  localStorage.setItem(_pbKey(exerciseName, gymIndex), JSON.stringify(data));
}
```

Also update the `module.exports` line at the bottom of `storage.js` to export them:

### Task 2: Add `checkPB()` to `app.js` and wire it into the done transition

**Files:**
- Modify: `app.js`

No new test file — `app.js` has no test file and directly manipulates the DOM. The logic branch is simple enough to verify visually.

- [ ] **Step 1: Add `checkPB()` function to `app.js`**

Add the following function to `app.js`, right before the `// ── Workout Completion ─────────────────────────────` comment:

```js
// ── PB Detection ───────────────────────────────────
function checkPB(exerciseName, gymIndex) {
  const weight = getWeight(exerciseName, gymIndex);
  if (weight.value === '' || weight.value === null) return false;

  const pb = getPB(exerciseName, gymIndex);
  if (!pb) {
    setPB(exerciseName, gymIndex, { weight_raw: weight.value, weight_type: weight.type });
    return true;
  }
  if (weight.type !== pb.weight_type) return false;
  if (weight.value > pb.weight_raw) {
    setPB(exerciseName, gymIndex, { weight_raw: weight.value, weight_type: weight.type });
    return true;
  }
  return false;
}
```

- [ ] **Step 2: Wire `checkPB()` into `onStopwatchTap()` on the done transition**

In `app.js`, find the block inside `onStopwatchTap()` that handles `next.phase === 'done'`:

```js

  }
```
